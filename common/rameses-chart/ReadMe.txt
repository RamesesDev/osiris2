
NameSpace: "http://com.rameses.graph"

Graph Attributes:

value - List of IChartItem objects
type - the type of the graph to be rendered, values are: line, pie, bar.
xvalue - String value used as label for the x-axis;
yvaue - String value used as label for the y-axis;
topCaption - String value used as top caption of the graph image;
bottomCaption - String value used as bottom caption of the graph image;

Note:
-This library depends on jfreechart-1.0.8.jar and jcommon-1.0.12.jar

-To make this Graph Tag work, graph.renderer.ChartServlet servlet must be registered on
 the web.xml of the project.
